# 华为 Mate 系列手机强制降分辨率
## 事件经过
### 2019.10
Mate30 用户拿到手机后发现在玩游戏的时候，没有经过用户的同意华为就把分辨率降低了，且无法调整为正常的分辨率。
### 2019.11
系统更新解决了该问题。
