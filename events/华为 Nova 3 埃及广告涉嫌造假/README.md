# 华为 Nova 3 埃及广告涉嫌造假
## 事件经过
Nova 3i 打的是 AI 牌，这段广告本想展示 AI 相机照片的处理技术水平高，素颜上镜也能一键美颜。

广告发布的第二天，事情发生了转折。

视频中的女演员 Sarah Elshamy 在社交平台 Instagram 中放出了一张照片。照片显示，拍摄那张素颜变全妆的照片时，男演员手中空空无手机。

广告中女主角化妆一般的 AI 拍摄效果，实则出自旁边拿着单反相机的专业摄影师之手。
